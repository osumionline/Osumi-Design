OBadge

OBadge es una directiva que permite mostrar un pequeño indicador de estado en elementos de la interfaz de usuario. Se puede usar para mostrar un estado o un conteo de elementos y permite ser usado no solo en componentes de Osumi Design, sino en cualquier tipo de contenido HTML.

El contenido del badge se actualiza en tiempo real y se muestra en caso de que su valor sea 0.

Referencia de API para OTabs

import { OBadgeDirective } from '@osumi/design';

OBadgeDirective: Directiva que muestra el badge.

Propiedad Descripción
@Input()
value: string
 Valor del badge.
