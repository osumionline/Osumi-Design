OTabs

OTabs es un componente que permite mostrar contenido en pestañas. Cada pestaña tiene un título y un contenido asociado.

Eventos

El componente OTabs emite un evento cuando se selecciona una pestaña. El evento es de tipo number y contiene la pestaña seleccionada.

Referencia de API para OTabs

import { OTabsComponent, OTabComponent } from '@osumi/design';

OTabsComponent: Componente que contiene las pestañas.

Propiedad Descripción
@Output()
selectedTabChange: number
 Evento lanzado al cambiar de pestaña. Contiene el índice de la pestaña seleccionada.

OTabComponent: Componente para alojar el contenido de una pestaña concreta.

Propiedad Descripción
@Input()
label: string
 Título de la pestaña.
